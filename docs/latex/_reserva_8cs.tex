\doxysubsection{Descrição detalhada}
Representa uma reserva feita por um cliente para um alojamento. 

Esta classe armazena informações sobre uma reserva, incluindo o cliente, o alojamento, as datas de início e fim, o preço total, e o cálculo do número de noites. Ela também é responsável por gerar um ID único para cada reserva.
