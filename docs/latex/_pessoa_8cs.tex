Definição da classe Pessoa para representar informações pessoais. 

Este ficheiro contém a implementação da classe Pessoa, que armazena informações sobre uma pessoa, como nome, data de nascimento, identificação, telefone e email. Inclui também métodos para calcular a idade e exibir informações.
