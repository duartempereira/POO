A classe \doxyref{Gestor\+Reservas}{p. }{class_gestao_alojamentos_turisticos_1_1_gestor_reservas} é responsável por gerir as reservas dos clientes, permitindo adicionar, procurar, verificar a disponibilidade de alojamentos e listar todas as reservas realizadas.
